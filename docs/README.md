#### 数据结构这门学科有什么用 ？

首先，现实世界中的问题大多属于 「非数值计算」 的问题，</br>

通过学习数据结构这门学科，使我们能够将这些问题信息化，然后将信息交由计算机进行高效处理，最终解决问题，创造价值。</br>

大致流程为：

  分析问题 => 挑选或构造合适的数据结构 => 设计并编写程序 => 计算机运行程序 => 问题解决创造价值
