#### 基本概念

> 线性表 + 只允许在一端进行插入/删除操作 => 栈（Stack）

<img src="https://gitee.com/pj-l/imgs-1/raw/master/screenShot/image-20211012151723387.png"></img>

#### 基本操作

> 创、销、增删改查

```cpp
InitStack(&S);    // 初始化一个栈对象
DestroyStack(&S); // 销毁一个栈对象

Push(&S, x);      // 进栈
Pop(&S, &x);      // 出栈

GetTop(S, &x);    // 读栈顶
StackEmpty(S);    // 栈判空
```
