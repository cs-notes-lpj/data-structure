中缀表达式 => 后缀表达式（逆波兰式）=> algorithms/normal-2-reverse-polish-notation.c

```txt
  目前只支持操作数是个位数
    输入：((5/(7-(1+1)))*3)-(2+(1+1))
    输出：5 7 1 1 + - / 3 * 2 1 1 + + -

  待尝试操作数有多位数的情况
    输入：((15/(7-(1+1)))*3)-(2+(1+1))
    输出：15 7 1 1 + - / 3 * 2 1 1 + + -
```

有几种思路：

  1. 用单链表存，而不用数组存
  2. 建立表达式树，然后中序遍历
  3. 字符转整型，统一用 ascii 码存；即 int 数组
  4. 将 12 存成 '1' 和 '2', 统一用 char 型数组存，只不过在数组中添加空格，用来分割操作数和操作符
  5. 用指针型数组，即数组元素都是一个个的地址，这些地址有可能指向的是数，有可能指向的是字符
